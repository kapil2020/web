DRUM

Dynamic Route Planning for Urban Mobility

Live Demo GitHub

Project Overview

DRUM is a comprehensive web application that revolutionizes urban route planning by incorporating environmental and sustainability metrics. Unlike traditional navigation apps that optimize solely for time or distance, DRUM provides four distinct routing modes:

LEAP Mode

Least Exposure Airway Path - Minimizes PM2.5 exposure

LECR Mode

Least Energy Consumption Route - Optimizes energy use

Technical Implementation

Frontend

  • React.js with TypeScript
  • Leaflet.js for mapping
  • Tailwind CSS

Backend

  • Python Flask API
  • PostgreSQL with PostGIS
  • Redis caching
